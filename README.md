# Interpretable ESG Modeling using Hybrid TP–ANFIS Framework with Dynamic Rule Obsolescence (DRO)

## Overview
This repository provides the data, models, and implementation supporting a hybrid interpretable AI framework for ESG-based sustainability assessment under uncertainty.

The framework integrates Tensor-Product (TP) modeling, ANFIS, and a Dynamic Rule Obsolescence (DRO) mechanism to balance interpretability and predictive performance.

## Key Contributions
- Hybrid TP–ANFIS interpretable modeling framework
- Compact rule extraction using tensor decomposition
- Comparative analysis of ANFIS vs TP under rule constraints
- Dynamic Rule Obsolescence (DRO) for temporal rule stability

## Repository Structure
- `/data` – ESG datasets and processed data
- `/code` – MATLAB and Python implementations
- `/models` – Trained models and rule sets
- `/results` – Output data and evaluation results
- `/figures` – Figures used in the manuscript
- `/docs` – Supporting documentation

All datasets and code necessary to reproduce the results are included.
